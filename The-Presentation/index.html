What is Plyr?

Plyr is an HTML5 video player written mostly in JavaScript and CSS. It's design is both beautiful and functional. It's also very user friendly and easy to implement!

Show me!

Let's look at the way it behaves and how it handles videos.

Who is the developer of Plyr?

Plyr was made by Sam Potts and first pushed to GitHub in February, 2015. Other developers have also helped, but Sam has made more than 95% of the commits.

Installation

Installing Plyr is as simple as downloading it from GitHub and including the files from the dist directory.

Next, throw some video tags in a div with the class player. After doing that, call plyr.setup() optionally passing an object with configuration.

That's all there is to do!
