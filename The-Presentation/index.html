What is Plyr?

Plyr is an HTML5 video player written mostly in JavaScript and CSS. It's design is both beautiful and functional. It was created this February (2015) by Sam Potts who is a developer for Selz, an ecommerce company in Syndey! You can find Plyr over at GitHub.

Show me!

That's enough chatting. Let's see it live!

Okay that's neat. How did you do that, though?

Implementing Plyr couldn't be more simple. Download the Plyr repository and move the three files in the dist directory to your asset directory.

Now you're ready to start Plyr! There are just a few more things you need to do.

Final Set-up of Plyr

All you'll need to do is import both dist/plyr.js and dist/plyr.css then make a call to plyr.setup() in a script element. You can also pass an object to plyr.setup() with configuration settings.

Next, throw a video tag into a div with the class player. After doing that, call plyr.setup() and the self-executing AJAX call to load Plyr's icons.

One More Thing

I've also used a second plug-in which is called Vide. It allowed me to achieve the lovely background video without putting a lot of effort and time into the development necessary to bring it to life.

It was as easy as downloading Vide then including both jQuery and dist/jquery.vide.min.js. After that I'd set the data- attribute data-vide-bg to my body object with a value of the path (and name) of my video.
